What are HTML and CSS used for?

You can think of HTML as the skeleton that gives every webpage structure.

Think of it like skin and makeup that covers the bones of HTML.

HTML stands for HyperText Markup Language.

Things inside < > are called tags

Tags nearly always come in pairs: an opening tag and a closing tag.

Example of opening tag: <html>
Example of closing tag: </html>

There are always two parts to an HTML file: the head and the body.

The head contains information about your HTML file, like its title.

The body is where you put your content, such as text, images, and links.

Type in the following...

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Name of your page</title> </head> <body> </body> </html>

Create all types of headings <h1> to <h6> and with paragraphs <p>... Don't forget to close the tags.

Putting the Link into HTML

Hypertext means "text with links in it."

<a href="http://sendMeThere.com"></a>

Image tags

img tags have / in the tag to close it:

<img src="url" />

Construct a linked image?

<a href="url"><img src="url" /></a>

Ordered and Unordered Lists

Want to make some lists?

<h1>List of my favorite things</h1> <ol> <li>raindrops on roses</li> <li>whiskas on kittens</li> <li>call of duty: modern warfare</li> </ol> <h2>List of things I find ok</h2> <ul> <li>the sound of silence</li> <li>salzburg</li> <li>stuff</li> </ul>

In-line CSS

We can give tags more instructions by including attributes in the opening tag.

<h2 style="color:red"> <h2 style="color: green; font-size:12px"> <h1 style="font-family: Arial"> <p style = "font-size:14px; color: orange; font-family: Bodoni; text-align:center">

Background colo(u)r

<body style="background-color: black; color:white"> <h3>Favorite Football Teams</h3> <ol style="background-color: white; color:green"> <li>The Collingwood Football Club</li> <li>Werder Bremen</li> <li>none</li> </ol> </body>

Tables

You don't really create columns in <table>s: instead, you tell each row how many cells to have, and that determines your number of columns

King Kong 1933
Dracula 1897
Bride of Frankenstein 1935
<table border="1px"> <tr> <td>King Kong</td> <td>1933</td> </tr> <tr> <td>Dracula</td> <td>1897</td> </tr> <tr> <td>Bride of Frankenstein</td> <td>1935</td> </tr> </table>

Divs

Short for "division," <div> allows you to divide your page into containers (that is, different pieces). This will come in handy when you begin learning CSS

<div style="height:50px; background-color:red"></div> <!-- NOTE Missing width:50px; --> <div style="width:50px; height:50px; background-color:blue"></div> <div style="width:50px; height:50px; background-color:green"></div> <div style="width: 50px; height:50px; background-color: yellow"></div>
